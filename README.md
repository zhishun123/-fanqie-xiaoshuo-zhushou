# 🍅 番茄小说自动发布系统 (Fanqie Auto Publisher)

一个基于 Playwright 和 PyQt5 构建的番茄小说自动化上传工具。它可以自动解析本地文件夹中的 TXT 章节文件，并模拟人工在番茄小说创作者后台自动完成发布，支持智能填写定时发布时间、自动勾选 AI 选项以及跳过风险检测。

## 🌟 核心特性

- **可视化面板 (GUI)**：提供了一个极客风（Catppuccin 深色主题）的 PyQt5 客户端面板。
- **自动断点续传**：支持设置起始章节与结束章节，防止重复上传。
- **章节智能解析**：自动读取 `第XX章_标题.txt` 格式文件，并剥离正文首行的重复标题。
- **完整交互模拟**：
  - 自动跳过“错别字检测”
  - 自动跳过“风险词汇检测”
  - 自动勾选“使用AI”
- **高级定时发布**：
  - 完美破解并兼容 Arco Design 框架的只读日期时间选择器
  - 支持首发时间设定与分钟级间隔（如：每 60 分钟发布下一章）
- **会话保持**：初次扫码登录后，后续任务免登录运行。
- **线程分离架构**：使用子进程解决 Playwright 异步事件循环冲突，保证上传期间 GUI 流畅无卡顿。

## 📦 安装说明

1. 确保已安装 Python 3.8+。
2. 克隆本仓库：
   ```bash
   git clone https://github.com/yourusername/fanqie-auto-publisher.git
   cd fanqie-auto-publisher
   ```
3. 安装依赖包：
   ```bash
   pip install -r requirements.txt
   ```
4. 安装 Playwright 内置浏览器：
   ```bash
   playwright install chromium
   ```

## 🚀 运行方式

### 方式一：GUI 图形化界面（推荐）
这是最高效、体验最好的操作模式。

```bash
python gui.py
```
运行后，你可以在界面中直观地设置作品 URL、本地文件夹路径、定时发布时间等，并一键开启任务。

### 方式二：CLI 命令行终端
如果你更喜欢在终端工作：

```bash
python main.py
```
终端会提供互动菜单：
1. **解析预览**：检查你的 TXT 文件是否被正确识别。
2. **自动上传**：开始批量上传流程。
3. **单章测试**：只上传第一章用于测试。
4. **登录保存**：单独打开浏览器以供扫码登录。

## ⚙️ 配置文件说明 (`config.py`)

你可以直接通过 `gui.py` 界面进行修改，也可以手动编辑 `config.py` 文件：
- `WORK_URL`: 你的番茄作品后台链接
- `CHAPTERS_DIR`: 存放 TXT 格式小说的绝对路径
- `SCHEDULED_PUBLISH`: 是否开启定时发布
- `ACTION_DELAY`: 每个动作间的间隔（秒），建议保持在 1.5 秒以上防风控
